# Планируемые цели

## Сеть Host -> Ruby
- Локальный транспорт: UDP на localhost (чтобы Ruby не подвисал на TCP).
- Batch: каждые 16 мс хост формирует один пакет BatchState со всеми игроками и отправляет в Ruby.
- Heartbeat/таймаут: если LastSeenUtc старше 300–500 мс, сбросить keysMask = 0.

## Ввод
- Пока не расширять keysMask; используем текущий 64-битный формат.

seq: если хочешь игнорировать “старые” пакеты — храни last_seq и не принимай меньше/равно.
